## Explique o funcionamento da web, desde a solicitação de um recurso num servidor web até a exibição na tela do navegador. 
Quando acessamos um navegador, ou melhor, uma ferramenta de pesquisa, é feita uma requisição para um Domain Name Server(DNS). Essa request caso não esteja armazenada em cache retorna um endereço de IP do server que providencia tal página, usando o protocolo HTTPS, o server então, procura o diretório da request e o retorna pro navegador a aplicação, ou a página estática.
